# Explorating Shakespeare and Company Project data

This folder includes a Jupyter notebook for guided exploration of data from the [Shakespeare and Company Project](https://shakespeareandco.princeton.edu/), focused on library member borrowing behaviors.

This work is drawn from work for the article "Missing Data, Speculative Reading", by Rebecca Sutton Koeser and Zoe LeBlanc. (Read the [article on *Modernism/modernity PrintPlus*](https://modernismmodernity.org/forums/posts/koeser-leblanc-missing-data-speculative-reading); read [article on *Journal of Cultural Analytics*](https://culturalanalytics.org/article/116926-missing-data-speculative-reading).)
